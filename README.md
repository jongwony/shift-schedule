# Nurse Shift Scheduling Feasibility Checker

간호사 근무표의 제약 조건 충족 여부를 실시간으로 검사하는 도구입니다.

## 기능

- **직원 관리**: 직원 추가/삭제, 주휴일 설정
- **근무표 편집**: 28일(4주) 단위 근무표 작성 (D/E/N/OFF)
- **제약 조건 검사**: 6가지 제약 조건 실시간 검증
  - 역순 금지 (D-E-N 순서)
  - N-OFF-D 금지
  - 연속 나이트 제한
  - 월간 나이트 요구량
  - 최소 인원 요구
  - 주휴 일치
- **내보내기/불러오기**: JSON 형식으로 데이터 저장 및 복원

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 빌드

```bash
npm run build
```

빌드 결과물은 `dist/` 디렉토리에 생성됩니다.

### 테스트

```bash
npm test
```

## 사용법

1. **직원관리 탭**: 직원 이름과 주휴일을 설정하여 직원 추가
2. **근무표 탭**: 시작일 선택 후 각 셀을 클릭하여 근무 배정
3. **설정 탭**: 제약 조건 파라미터 조정 (연속 나이트 최대일수, 최소 인원 등)
4. 화면 하단에서 실시간 타당성 검사 결과 확인
